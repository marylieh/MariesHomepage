Hi! I'm Marie

Developer Designer Gamer

Hey, my name is Marie! My main Programming Language is Java. With Java I code Plugins for the video game "Minecraft". For example, these can allow players to create "warp points". The current coordinates of the player are saved in these points. Later the players can teleport back to the corresponding warp point. Another example would be "SimplePerms". With this plugin permissions can be managed and groups can be created. So different permissions can be assigned to different players. So that not every player is able to do everything on the server. Groups are used to manage multiple players. For example, several players can have the same permissions without having to set the permissions for each player individually.
